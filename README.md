# Katakana FSRS

A spaced repetition system (SRS) app for learning Japanese Katakana characters using the FSRS algorithm.

## Features

- Learn all 46 basic Katakana characters
- Spaced repetition using the FSRS algorithm
- Speed-based grading system
- Progress tracking and statistics
- Local storage persistence

## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start learning Katakana!

## How It Works

The app shows you a Katakana character and you type the correct romaji (e.g., for "ã‚¢" you type "a").

The FSRS algorithm intelligently schedules reviews based on your performance, helping you learn efficiently.
